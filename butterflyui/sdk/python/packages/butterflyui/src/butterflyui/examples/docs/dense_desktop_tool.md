# Dense Desktop Tool Example

File: [dense_desktop_tool.py](G:/Projects/ButterflyUI/butterflyui/sdk/python/packages/butterflyui/src/butterflyui/examples/dense_desktop_tool.py)

This example shows how the Styling engine applies to a dense tool-style
desktop shell.

It demonstrates:

- sidebar and tab styling through shared classes
- panels and work areas using the same token model as buttons and text
- compact layouts that still rely on the Styling engine rather than local paint
- a practical desktop-app composition pattern

Run:

```bash
python -m butterflyui.examples.dense_desktop_tool --target desktop
```

Use it when you want to build IDE-like or tool-heavy applications.

# Landing Page Example

File: [landing_page.py](G:/Projects/ButterflyUI/butterflyui/sdk/python/packages/butterflyui/src/butterflyui/examples/landing_page.py)

This example is the public proof page for the Styling engine.

It demonstrates:

- stylesheet selectors driving the hero shell
- utility classes on navigation, buttons, pills, and cards
- direct-authored scene layers from Python
- token-aware typography and spacing
- responsive layout authored with ordinary ButterflyUI controls

Run:

```bash
python -m butterflyui.examples.landing_page --target desktop
```

Use it when you want a marketing-page or landing-page baseline that shows how
ButterflyUI styling replaces one-off visual controls.

## What Genesis proves

Genesis uses Styling for:

- shell-wide galaxy-like authored scene layers
- shared tokens and a generated stylesheet
- buttons, cards, drawers, tabs, sidebars, and inputs
- overlay and drawer chrome
- dark-shell theming with runtime accent changes

That makes Genesis the best current proof that ButterflyUI Styling can power a
real tool-style application rather than only isolated component demos.

## Why this fixture matters

The Styling engine should not only make a landing page look good. It should
also support a long-lived desktop-style interface with:

- sidebars
- tabs
- model drawers
- overlays
- settings panels
- interactive inputs

Genesis is where those pieces meet the real engine.

# Dashboard Example

File: [dashboard.py](G:/Projects/ButterflyUI/butterflyui/sdk/python/packages/butterflyui/src/butterflyui/examples/dashboard.py)

This example demonstrates a dense analytical shell with Styling-first panels.

It demonstrates:

- dark token packs
- sidebar, toolbar, and panel layout driven by stylesheet classes
- metric cards and surfaces using shared token roles
- authored scene layers living behind a real app shell
- a layout pattern closer to a production dashboard than a hero page

Run:

```bash
python -m butterflyui.examples.dashboard --target desktop
```

Use it when you want a reference for data-heavy application chrome.

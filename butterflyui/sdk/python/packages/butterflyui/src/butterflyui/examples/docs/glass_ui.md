# Glass UI Example

File: [glass_ui.py](G:/Projects/ButterflyUI/butterflyui/sdk/python/packages/butterflyui/src/butterflyui/examples/glass_ui.py)

This example focuses on translucent surfaces, authored atmospheric layers, and
token-driven frosted shells.

It demonstrates:

- backdrop blur with Styling-first surfaces
- authored scene layers clipped into glass shells
- tokenized borders, radii, and text roles
- translucent CTA/button styling through stylesheet classes

Run:

```bash
python -m butterflyui.examples.glass_ui --target desktop
```

Use it when you want to build premium frosted or ambient interfaces.

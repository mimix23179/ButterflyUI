# New Controls Showcase Demo

This demo exercises the newly added/returned controls in one screen:

- `Card`
- `DataGrid`
- `Skeleton`, `SkeletonLoader`
- `Divider`
- `ParticleField`, `ScanlineOverlay`, `Vignette`
- `Surface`, `Box`, `Row`, `Column`, `Stack`, `Wrap`, `ScrollView`
- `PromptComposer`

## Run

From the repo root:

```powershell
G:/Projects/ButterflyUI/env/Scripts/python.exe demo/new_controls_showcase/main.py
```

If you prefer module-style execution, ensure the ButterflyUI package is installed in your environment.

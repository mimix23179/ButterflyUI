# New Controls Showcase Demo

This demo exercises the newly added/returned controls in one screen:

- `Card`
- `DataGrid`
- `Skeleton`, `SkeletonLoader`
- `Divider`
- `ParticleField`, `ScanlineOverlay`, `Vignette`
- `Surface`, `Box`, `Row`, `Column`, `Stack`, `Wrap`, `ScrollView`
- `PromptComposer`
- `Align`, `BoundsProbe`
- `ArtifactCard`, `AttachmentTile`
- `AsyncActionButton`
- `Audio`
- `BarPlot`
- `BottomSheet`
- `BrushPanel`
- `Canvas`

## Run

From the repo root:

```powershell
G:/Projects/ButterflyUI/env/Scripts/python.exe demo/ButterflyUI_Showcase/main.py
```

If you prefer module-style execution, ensure the ButterflyUI package is installed in your environment.

```powershell
G:/Projects/ButterflyUI/env/Scripts/python.exe -m demo.ButterflyUI_Showcase.main
```
